# Roadmap

## Recently shipped

- standard label sync with a managed label manifest
- beginner, package, docs, and security-sensitive presets
- 6 public example repositories that show what each preset produces
- public case study derived from the `ShuleDocs` adoption work
- `check-docs` for local Markdown link and anchor validation

## Next up

- Add a dry-run diff mode so maintainers can preview generated changes.
- Add more presets for app frameworks and deployment targets.
- Add richer social-preview and README assets for each public example repository.
- Add a release-note schema option for teams that want structured outputs.
- Add framework-specific service presets, starting with common web-app and API shapes.
